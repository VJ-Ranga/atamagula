Step 1

Set Your Home Page URL

Open whichever pattern file she chose (e.g. glass-a.html) and find line 37 at the very top of the script:

const HOME = 'YOUR_HOME_PAGE_URL';   // ← change this

Replace it with the real URL:

const HOME = 'https://yourwebsite.com/';   // ✅ done

Step 3 — Show Only Once Per User

One Time Per Session

Add these two snippets to the glass file. First, paste this right after const HOME = '...'; :

// Skip intro if already seen this session
if(sessionStorage.getItem('intro_seen')){
  location.replace(HOME);
}

Second, inside the fade() function, add one line before document.body.appendChild(ov); :

function fade(){
  sessionStorage.setItem('intro_seen','1'); // ← add this line
  const ov=document.createElement('div');
  ...

Now the intro plays once. If the user opens a new tab or comes back in the same browser session, it skips straight to the home page.

One Time Ever (Persists Across Sessions)

Same as above but use localStorage instead of sessionStorage:

if(localStorage.getItem('intro_seen')){ location.replace(HOME); }

// inside fade():
localStorage.setItem('intro_seen','1');

This remembers even after the browser is closed. The intro never plays again for that user on that device.

Tip To reset during testing, open DevTools → Application → Local Storage → delete the intro_seen key.
